# 雷军视角 Skill - 运营维度深化说明

## 更新概览

本次更新深度整合了**雷军的运营方法论**，将Skill从"思维模拟"升级为"思维+运营"双引擎。

## 核心增强

### 1. 新增运营心智模型

在原有7个战略心智模型基础上，增加了**运营决策框架**：

- **流量vs口碑权衡**：决策树模型
- **七大运营策略**：话题制造、自嘲公关、参与感运营、演讲即产品、价格锚点、长期IP、全员运营
- **多平台运营矩阵**：微博/抖音/直播/B站的差异化策略
- **运营禁忌**：什么坚决不做

### 2. 新增案例库

创建了`operation-cases.md`，收录17个经典案例：

**危机公关类**：
- "Are you OK"从黑历史到文化符号
- 小米汽车"像保时捷"争议营销
- 小米10定价争议应对

**话题制造类**：
- 雷军年度演讲IP化运营
- "One More Thing"惊喜制造
- 小米SU7价格揭晓戏剧化

**用户共建类**：
- MIUI"橙色星期五"
- 米粉节运营

**直播带货类**：
- 雷军直播卖车
- 卢伟冰"价格屠夫"人设

### 3. 新增方法论文档

创建了`operation-methodology.md`，系统化整理：

- **多平台运营策略**：各平台定位、内容类型、话术模板
- **运营心法**：话题制造、自嘲公关、参与感、演讲IP、价格锚点、长期IP、全员运营
- **运营禁忌**：5大坚决不做的事

## 典型使用示例

### 示例1：产品发布会策划
```
/leijun-perspective "我们要发布一款智能音箱，价格599元，主打音质，怎么做发布会？"
```

**会得到**：
- 完整的发布会结构（时长、环节、话术）
- 雷军式演讲脚本示例
- 价格揭晓策略（如何制造落差感）
- 多平台预热方案
- 度量标准

### 示例2：社交媒体内容
```
/leijun-perspective "帮我写5条产品预热微博，倒计时5天到发布"
```

**会得到**：
- 5条符合雷军风格的微博文案
- 配图建议
- 话题标签建议
- 互动引导设计

### 示例3：危机公关
```
/leijun-perspective "我们的产品被质疑抄袭某友商，雷军会怎么应对？"
```

**会得到**：
- 雷军式危机公关策略（自嘲转化法）
- 具体话术示例
- 时间线规划（1小时/6小时/24小时/1周）
- 什么能说、什么不能说

### 示例4：直播带货
```
/leijun-perspective "我要直播卖这款产品，帮我设计一个60分钟的直播脚本"
```

**会得到**：
- 完整的60分钟直播结构
- 开场/讲解/价格/下单各环节话术
- 与传统主播的差异化打法
- 转化率提升技巧

## 实现方法

### 知识来源
- `operation-methodology.md`：10章节系统方法论
- `operation-cases.md`：17个真实案例拆解
- `system-prompt.md`：整合到AI的决策逻辑

### 模型设计
```
用户输入（运营需求）
    ↓
识别问题类型（发布会/社交媒体/危机公关/直播等）
    ↓
选择适配策略（七大运营策略中选2-3个）
    ↓
生成方案（结构+话术+度量）
    ↓
风格校验（必须符合雷军表达DNA）
    ↓
输出（可直接执行的运营方案）
```

### 质量保证

每个运营回答必须包含：
- ✅ 具体可执行的动作（不要空泛建议）
- ✅ 雷军式话术示例（直接可用）
- ✅ 渠道选择理由
- ✅ 度量标准（如何衡量效果）
- ✅ 风险提示（什么不能做）

## 差异化优势

### 对比其他"雷军模拟"
| 维度 | 其他模拟 | 本Skill |
|------|---------|---------|
| 战略思维 | ✅ 有 | ✅ 7个心智模型 |
| 表达风格 | ✅ 有 | ✅ DNA级还原 |
| 运营方法 | ❌ 无或浅 | ✅ **系统化+案例化** |
| 话术库 | ❌ 无 | ✅ **分场景可套用** |
| 案例支撑 | ❌ 无 | ✅ **17个真实案例** |
| 可执行性 | ⚠️ 理论 | ✅ **直接可用方案** |

### 核心价值

1. **不只是"说得像"，更是"做得对"**
   - 不只是模仿雷军的表达风格
   - 更重要的是还原雷军的运营打法

2. **基于真实案例**
   - 每个策略都有对应的小米实战案例
   - "Are you OK"、小米SU7、年度演讲等

3. **可直接执行**
   - 提供完整的运营方案
   - 包含话术、结构、时间线、度量标准

4. **风险提示**
   - 明确什么不能做
   - 避免踩雷军踩过的坑

## 使用限制

### 不能做的事
❌ 编造雷军未说过的话
❌ 超出公开信息的内部决策细节
❌ 政治、敏感话题的评论
❌ 对竞品的负面攻击（违背雷军风格）

### 需要用户自己判断的
⚠️ 具体的预算分配（需要根据公司情况调整）
⚠️ 法律合规性（不同行业有不同限制）
⚠️ 团队执行能力（方案需要匹配团队资源）
