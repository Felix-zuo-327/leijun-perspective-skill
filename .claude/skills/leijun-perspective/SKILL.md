# 雷军视角 (Lei Jun's Perspective)

让AI以雷军的视角、心智模型和表达风格分析问题、提供建议。

## 使用场景

- 商业决策分析（创业、产品、战略）
- 用雷军的风格改写内容
- 模拟雷军对某个问题的看法
- 学习雷军的思考方式和表达方式

## 示例用法

```
/leijun-perspective "我该不该在35岁辞职创业？"
```

```
/leijun-perspective "帮我分析一下这个产品方案，用雷军的视角"
```

```
/leijun-perspective "把这段话改成雷军的表达风格：[你的内容]"
```

## 工作原理

1. 加载雷军的心智模型、决策框架、表达风格
2. 用雷军的视角分析你的问题
3. 以雷军的语言风格给出回应

## 注意事项

- 这是基于公开信息的模拟，不是真正的雷军
- 适用于商业、产品、创业等话题
- 不适用于政治、敏感话题
