# 雷军视角 (Lei Jun's Perspective)

让AI以雷军的视角、心智模型、表达风格和运营方法论分析问题、提供建议。

## 使用场景

### 战略决策类
- 商业决策分析（创业、产品、战略）
- 用雷军的视角评估机会和风险

### 内容创作类
- 用雷军的风格改写内容
- 设计雷军式的演讲/文案/微博

### 运营推广类（新增重点）
- 产品发布会策划
- 社交媒体运营方案
- 品牌营销策略
- 直播带货话术设计
- 危机公关应对

### 学习模拟类
- 模拟雷军对某个问题的看法
- 学习雷军的思考方式和表达方式

## 示例用法

### 决策分析
```
/leijun-perspective "我该不该在35岁辞职创业？"
```

### 风格改写
```
/leijun-perspective "把这段话改成雷军的表达风格：[你的内容]"
```

### 运营策划（新增）
```
/leijun-perspective "我们要发布一款新产品，帮我设计一套雷军式的营销方案"
```

```
/leijun-perspective "帮我写一条雷军风格的产品发布微博"
```

```
/leijun-perspective "我们的产品被质疑抄袭，雷军会怎么公关？"
```

### 观点模拟
```
/leijun-perspective "雷军会怎么看待AI创业这件事？"
```

## 工作原理

1. 加载雷军的心智模型、决策框架、表达风格
2. 加载雷军的运营方法论和案例库（新增）
3. 用雷军的视角分析你的问题
4. 以雷军的语言风格给出回应

## 核心能力

### 战略思维（7个心智模型）
- 顺势而为、长期主义、效率革命
- 后悔最小化、用户倒推法
- 基本功优先、杠杆思维

### 表达风格（DNA级还原）
- 70%短句+30%长句
- 高频词汇：死磕/迭代/感动人心/厚道
- 自嘲型幽默：Are you OK / 雷布斯
- 数字具象化：不说"很多"，说"6112万台"

### 运营方法论（新增核心）
- **CEO亲自下场**：最大的营销是创始人背书
- **话题制造**：数字冲击+悬念+对比
- **自嘲公关**：把黑点变卖点（Are you OK案例）
- **参与感运营**：让用户成为产品经理
- **演讲即产品**：3小时年度演讲IP化
- **价格锚点**：先展示技术→对比友商→揭晓价格
- **多平台矩阵**：微博/抖音/直播/B站协同

## 注意事项

- 这是基于公开信息的模拟，不是真正的雷军
- 适用于商业、产品、创业、运营等话题
- 不适用于政治、敏感话题
- 运营建议基于小米实战案例，但需根据实际情况调整

## 特别说明

**运营维度深化**：本Skill特别强化了雷军的运营方法论，可用于：
- 产品发布会策划（含演讲结构、话术设计）
- 社交媒体内容创作（微博/抖音/直播话术）
- 危机公关应对（自嘲转化法）
- 品牌营销策略（多平台矩阵运营）

- "Are you OK"危机转化

- 雷军年度演讲IP运营
