# 运营维度深化 - 快速测试

## 测试目标
验证雷军视角Skill的运营模块是否正常工作

## 测试用例

### 测试1：产品发布会策划
**输入**：
```
/leijun-perspective "我们要发布一款无线耳机，价格399元，主打降噪，怎么做发布会？"
```

**预期输出应包含**：
- ✅ 完整的发布会结构（时长、环节）
- ✅ 雷军式演讲话术示例
- ✅ 价格揭晓策略
- ✅ 多平台预热方案（微博/抖音/直播）
- ✅ 度量标准（预约量/首销时间/话题阅读）
- ✅ 风险提示（什么不能做）

### 测试2：社交媒体文案
**输入**：
```
/leijun-perspective "帮我写一条产品预热微博，明天发布新手机"
```

**预期输出应包含**：
- ✅ 符合雷军风格的微博文案（70%短句+30%长句）
- ✅ 高频词汇：死磕/迭代/感动人心等
- ✅ 悬念制造："大家猜猜..."类句式
- ✅ 数字具象化
- ✅ 配图建议
- ✅ 话题标签建议

### 测试3：危机公关
**输入**：
```
/leijun-perspective "我们的产品被质疑偷工减料，雷军会怎么回应？"
```

**预期输出应包含**：
- ✅ 雷军式公关策略（先接受后解释）
- ✅ 具体话术："大家说得有道理，但..."
- ✅ 成本透明化策略
- ✅ 时间线（1小时/6小时/24小时回应节奏）
- ✅ 什么能说、什么不能说

### 测试4：直播带货脚本
**输入**：
```
/leijun-perspective "我要直播卖小米平板，给我一个30分钟的直播脚本"
```

**预期输出应包含**：
- ✅ 完整的30分钟直播结构
- ✅ 开场/讲解/价格/下单各环节话术
- ✅ 技术讲解话术（不是"买买买"）
- ✅ 与传统主播的差异化
- ✅ 转化引导策略

### 测试5：风格改写（验证原有功能未受影响）
**输入**：
```
/leijun-perspective "把这段话改成雷军风格：我们的产品非常优秀，功能强大，值得购买"
```

**预期输出应包含**：
- ✅ 短句拆解
- ✅ 疑问句引导
- ✅ 高频词汇替换
- ✅ 具象化表达
- ✅ 雷军式结尾

## 验证标准

### 必须符合的风格特征
- [ ] 70%短句 + 30%长句
- [ ] 使用高频词汇（死磕/迭代/感动人心/厚道等）
- [ ] 数字具象化（不说"很多"，说具体数字）
- [ ] 低确定性表达（"我觉得...""可能..."）
- [ ] 自嘲型幽默（如果场景合适）
- [ ] 不使用禁忌词（垄断/打败/完美/唯一）

### 运营模块特有特征
- [ ] 提供多平台协同方案（微博+抖音+直播等）
- [ ] 包含具体可执行的话术示例
- [ ] 提供度量标准（如何衡量效果）
- [ ] 明确风险提示（什么不能做）
- [ ] 基于真实案例（提到Are you OK/小米SU7等）
